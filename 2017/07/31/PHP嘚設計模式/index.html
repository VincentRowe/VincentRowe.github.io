Factory Method（工廠方法模式）

Factory Method : UML

  • Product（產品）：定義Factory Method所造物件的介面
  • ConcreteProduct（具体产品角色）：具體實作出Product介面
  • Creator（創造）：宣告Factory Method，它會傳回Product類別之物件
  • ConcreteCreator（具體工廠）：複寫Factory Method以傳回ConcreteProduct的物件個體
